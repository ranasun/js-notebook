# JS Notebook

A jupyter notebook clone for javascript.

- Supports import statements eg: `import React from 'react'`
- Uses esbuild to bundle in the browser

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
npm run build
npm run preview
```

## Test

```bash
npm run test
npm run cypress:open
```
